経験とスキルと熱意を活かして酵素開発を推進する仲間を募集しています

株式会社digzymeでは、経験とスキルと熱意を活かして酵素開発を推進する仲間を募集しています。

01
サイエンスで社会課題に向き合う
バイオインフォマティクスを武器に、食品・化学・医薬・繊維業界の課題を解決します。
02
最前線の研究環境
NEDO採択案件をはじめ、産業界と密接に連携した酵素開発プロジェクトに従事できます。
03
スタートアップで成長する
少数精鋭のチームで裁量を持ち、事業の成長を自分ごととして推進できます。

DRY Researcher
バイオインフォマティクス研究員
▾
業務内容

酵素の探索・改良に向けたバイオインフォマティクス解析（配列解析・3D構造解析・機械学習モデル開発等）を担当いただきます。digzyme Moonlight™・Spotlight™の機能拡張にも関わっていただきます。

求めるスキル
  • 生命情報科学・計算生物学・タンパク質工学分野での研究経験
  • Python / Bashを用いたデータ解析・パイプライン開発
  • 配列相同性検索、系統解析、3D構造解析のいずれかの経験
  • 機械学習・深層学習の実装経験（歓迎）

WET Researcher
遺伝子工学・生化学研究員
▾
業務内容

バイオインフォマティクス解析でデザインした酵素遺伝子の実験的評価・機能改良を担当。大腸菌・酵母・枯草菌などの発現系を用いた組換えタンパク質発現、精製、酵素活性評価を行います。

求めるスキル
  • 遺伝子工学・タンパク質工学・生化学分野での実験経験
  • 組換えタンパク質発現・精製の実務経験
  • 酵素速度論的解析の経験
  • 大腸菌以外の発現系（酵母・枯草菌等）の経験（歓迎）
応募する →
Production Dev
製造プロセス開発研究員
▾
業務内容

酵素・化合物の大量製造プロセスの開発・スケールアップを担当。ラボスケールから中型・大型発酵槽への移行、精製プロセスの確立、製造コスト最適化に取り組んでいただきます。

求めるスキル
  • 発酵工学・バイオプロセス工学分野での実務経験
  • 培養スケールアップ・プロセス最適化の経験
  • GMP・品質管理に関する知識（歓迎）
  • 100L以上のスケールでの発酵経験（歓迎）
